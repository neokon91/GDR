# Consegna Nuovo DM

Questa pagina serve quando il vault viene dato a una persona che deve usarlo, non configurarlo.

## Apri Solo Queste Pagine

| Quando | Pagina |
| --- | --- |
| Primo avvio | [[Inizia Qui]] |
| Controllo iniziale | [[Risorse/Setup Guidato]] |
| Esempio completo | [[Demo - La Reliquia Spezzata]] |
| Preparare | [[1. DM Dashboard]] |
| Giocare | [[Durante il Gioco]] |
| Dopo la partita | [[Risorse/Post Sessione Guidato]] |
| Se qualcosa non torna | [[Risorse/Se Qualcosa Non Funziona]] |

## Cosa Ignorare

- Cartelle `z.*`.
- File con nomi tecnici.
- Impostazioni interne degli strumenti.
- Pagine di sviluppo, repository e automazioni.

## Cosa Non Toccare

| Area | Perche |
| --- | --- |
| `.obsidian` | Contiene impostazioni e strumenti gia pronti. |
| `z.automazioni` | Serve ai pulsanti e ai controlli. |
| `z.modelli` | Contiene i modelli con cui vengono create le note. |
| `z.bacheche` | Contiene bacheche operative gia collegate. |

## Primo Giro Consigliato

1. Apri [[Inizia Qui]].
2. Apri [[Risorse/Setup Guidato]] e controlla che sia tutto pronto.
3. Apri [[Demo - La Reliquia Spezzata]].
4. Apri [[Durante il Gioco]] per vedere come appare una sessione al tavolo.
5. Apri [[Risorse/Post Sessione Guidato]] per capire cosa succede dopo.

## Regola Pratica

Se devi giocare, usa le dashboard. Se devi capire il vault, usa le guide in [[Risorse/Risorse]]. Se una pagina sembra tecnica, puoi ignorarla.
