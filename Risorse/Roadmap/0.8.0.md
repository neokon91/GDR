# Roadmap 0.8.0

Obiettivo: completare il prodotto per una prima release ZIP, senza aprire ancora community o pubblicazione ufficiale.

## Priorita

- [x] Separare pubblicazione community da release ZIP iniziale.
- [x] Definire pipeline import Azgaar basata su export strutturati, non su `.map` diretto.
- [x] Aggiungere import GeoJSON Azgaar verso bozze di luoghi.
- [x] Integrare Hex Cartographer come vista per regioni e viaggi.
- [x] Integrare TTRPG Tools: Maps come vista mappa zoomabile per materiali al tavolo.
- [x] Rafforzare Media Extended con timestamp e scene.
- [x] Completare tabelle Dice Roller realmente utili al tavolo.
- [x] Definire quali plugin restano fuori dalla release base.
- [x] QA end-to-end su ZIP generato.

## Criterio Di Qualita

La release ZIP deve permettere a un DM non tecnico di aprire il vault, seguire la pagina introduttiva, usare la demo come test, importare materiali esterni come bozze e giocare senza leggere documentazione tecnica.
