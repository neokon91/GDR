# Roadmap 0.5.0

Obiettivo: rendere il vault uno strumento da worldbuilding tassonomico, dettagliato e connesso, adatto ad ambientazioni grandi.

## Completato

- [x] [[Atlante del Mondo]] come centro operativo per ambientazioni grandi.
- [x] Nuove aree: [[Mondi/Culture/Culture]], [[Mondi/Lingue/Lingue]], [[Mondi/Storia/Storia]], [[Mondi/Conflitti/Conflitti]], [[Mondi/Cosmologia/Cosmologia]].
- [x] Nuovi modelli guidati per cultura, lingua, era storica, conflitto e cosmologia.
- [x] Vista "Buchi Dell'Ambientazione" per trovare collegamenti mancanti.
- [x] [[Risorse/Worldbuilding Tassonomico]] per costruire mondi ricchi senza leggere documentazione tecnica.
- [x] Controllo qualità aggiornato per riconoscere le nuove categorie.

## Criterio Di Qualita

Un DM deve poter creare un'ambientazione grande partendo da [[Atlante del Mondo]], collegando luoghi, popoli, poteri, storia e misteri senza sapere cosa sia un campo tecnico.
