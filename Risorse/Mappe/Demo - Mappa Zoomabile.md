# Demo - Mappa Zoomabile

Usa questa nota per provare TTRPG Tools: Maps con una mappa grande, zoomabile e pronta al tavolo. I pin e i layer si modificano dal menu della mappa; la base visuale resta un file locale nel vault.

```zoommap
image: Risorse/Mappe/Asset/Demo - Valle di Brumafonda.svg
id: demo-valle-brumafonda
storage: note
width: 100%
height: 520px
minZoom: 0.35
maxZoom: 6
align: center
wrap: false
render: dom
```

## Uso Al Tavolo

- Apri la mappa quando i personaggi scelgono se passare dal ponte, aggirare il fiume o salire al santuario.
- Aggiungi marker solo per informazioni gia rivelate o sicure da mostrare.
- Per la [[Vista Giocatori]], duplica la nota e rimuovi segreti, aree nascoste e appunti GM.
