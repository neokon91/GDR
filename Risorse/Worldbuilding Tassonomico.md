# Worldbuilding Tassonomico

Questa guida serve a costruire ambientazioni grandi senza perdersi. Parti sempre da cose giocabili, poi aggiungi profondità.

## Ordine Consigliato

1. Crea un mondo.
2. Crea 2 o 3 grandi regioni.
3. Crea 2 culture riconoscibili.
4. Crea 1 lingua comune e 1 lingua antica o locale.
5. Crea 2 poteri in conflitto.
6. Crea 1 evento storico che spiega il presente.
7. Crea 1 clock o fronte che mostra cosa cambia se nessuno interviene.
8. Crea 1 mistero cosmologico o religioso.
9. Collega missioni, PNG e luoghi a questi elementi.

## Cosa Rende Ricco Un Mondo

| Area | Domanda utile |
| --- | --- |
| Geografia | Dove succede la storia? |
| Culture | Chi vive qui e cosa considera normale? |
| Lingue | Come si riconoscono popoli, classi sociali e antichi segreti? |
| Poteri | Chi comanda, chi resiste, chi trama? |
| Storia | Quale ferita del passato influenza il presente? |
| Religione | Cosa credono vero gli abitanti? |
| Cosmologia | Cosa esiste oltre il mondo visibile? |
| Conflitti | Cosa cambierà se i PG non intervengono? |
| Tracciati | Quale pressione sta avanzando in modo visibile? |

## Entita E Sottotipi

| Entita | Sottotipi utili |
| --- | --- |
| Luogo | insediamento, dungeon, rovina, tempio, regione naturale, regno, isola, continente |
| Fazione | gilda, culto, confraternita, ordine rituale, fazione generica |
| Cultura | popolo, etnia, tradizione |
| Storia | evento, conseguenza, leggenda, era, guerra, fondazione |
| Conflitto | crisi, rivalita, guerra |
| Tracciato | clock, progress track, fronte, rituale, minaccia, viaggio, progetto |

Quando una categoria non cambia il modo in cui giochi al tavolo, resta un campo `tipo`; quando cambia template, campi e viste, merita un template dedicato.

## Regola D'Oro

Ogni elemento importante deve avere almeno tre collegamenti:

- un luogo;
- un potere, cultura o persona;
- una conseguenza giocabile.
- se c'e pressione nel tempo, un tracciato attivo.
