# Matrice Strumenti

Questa pagina spiega quali strumenti Obsidian sono necessari al prodotto.

## Essenziali

| Strumento | Serve per |
| --- | --- |
| Dataview | Tabelle, dashboard, controlli e viste dinamiche. |
| Templater | Creazione guidata di note e contenuti. |
| Meta Bind | Pulsanti e campi interattivi. |
| JS Engine | Alcuni blocchi e logiche avanzate. |
| Metadata Menu | Campi guidati e modifica piu semplice. |
| Homepage | Apertura su `Inizia Qui`. |

## Consigliati

| Strumento | Serve per |
| --- | --- |
| Kanban | Bacheche di preparazione e post-sessione. |
| Iconize | Icone e leggibilita della struttura. |
| Folder Notes | Indici di cartella. |
| Style Settings | Regolazioni visuali. |
| Minimal Theme Settings | Aspetto coerente con tema Minimal. |

## Specifici TTRPG

| Strumento | Serve per |
| --- | --- |
| Dice Roller | Tiri rapidi e tabelle casuali. |
| 5e Statblocks | Schede creature. |
| Fantasy Statblocks | Compatibilita statblock fantasy. |
| Initiative Tracker | Supporto incontri. |
| Calendarium | Date del mondo e scadenze narrative. |
| Leaflet | Mappe interattive se configurate. |

## Opzionali

| Strumento | Serve per |
| --- | --- |
| Excalidraw | Mappe relazionali e schemi. |
| Advanced Tables | Editing tabelle piu comodo. |
| BRAT | Test di strumenti beta. |
