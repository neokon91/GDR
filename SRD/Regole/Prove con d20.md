# Prove con d20

Quando l'esito di un'azione e incerto, si effettua una prova con d20. Le prove con d20 includono prove di caratteristica, tiri salvezza e tiri per colpire.

## Procedura

| Voce | Riepilogo |
| --- | --- |
| 1 | Tira 1d20. Con vantaggio o svantaggio tira due d20 e usa rispettivamente il maggiore o il minore. |
| 2 | Aggiungi il modificatore di caratteristica rilevante, il bonus di competenza se applicabile e altri bonus o penalita. |
| 3 | Confronta il totale con la CD o la CA. Il tiro riesce se il totale e pari o superiore al bersaglio. |

## Classi Difficolta tipiche

| Voce | Riepilogo |
| --- | --- |
| Molto facile | CD 5 |
| Facile | CD 10 |
| Media | CD 15 |
| Difficile | CD 20 |
| Molto difficile | CD 25 |
| Quasi impossibile | CD 30 |

## Tiri salvezza

| Voce | Riepilogo |
| --- | --- |
| Forza | Resistere fisicamente alla forza diretta. |
| Destrezza | Schivare pericoli. |
| Costituzione | Sopravvivere a un pericolo tossico. |
| Intelligenza | Riconoscere un'illusione come falsa. |
| Saggezza | Resistere a un attacco psichico. |
| Carisma | Affermare la propria identita. |
