# Compilare la scheda del personaggio

Dopo le scelte iniziali si registrano privilegi di classe, tiri salvezza, abilita, percezione passiva, punti ferita, Dadi Vita, iniziativa, Classe Armatura, attacchi e incantesimi.

## Numeri principali

| Voce | Riepilogo |
| --- | --- |
| Tiri salvezza | Se competente, aggiungi bonus di competenza al modificatore di caratteristica appropriato. |
| Abilita | Se competente, aggiungi bonus di competenza al modificatore associato all'abilita. |
| Percezione passiva | 10 + modificatore di Saggezza (Percezione), inclusi modificatori applicabili. |
| Iniziativa | Usa il modificatore di Destrezza. |
| CA base | Senza armatura o scudo, 10 + modificatore di Destrezza. |
| Attacco in mischia | Modificatore di Forza + bonus di competenza, se competente e salvo proprieta diverse. |
| Attacco a distanza | Modificatore di Destrezza + bonus di competenza, se competente e salvo proprieta diverse. |
| CD incantesimo | 8 + modificatore da incantatore + bonus di competenza. |
| Attacco incantesimo | Modificatore da incantatore + bonus di competenza. |
^srd-compilare_scheda_personaggio-numeri-principali

## Punti ferita al 1º livello

| Classe | Punti ferita massimi |
| --- | --- |
| Barbaro | 12 + modificatore di Cos |
| Guerriero, paladino o ranger | 10 + modificatore di Cos |
| Bardo, chierico, druido, monaco, ladro o warlock | 8 + modificatore di Cos |
| Stregone o mago | 6 + modificatore di Cos |
