# Capacità di trasporto

La taglia e il punteggio di Forza determinano il peso massimo che una creatura può trasportare, trascinare, sollevare o spingere.

## Capacità di trasporto

| chiave | valore |
| --- | --- |
| Minuscola | For x 3,75 kg |
| Piccola/Media | For x 7,5 kg |
| Grande | For x 15 kg |
| Enorme | For x 30 kg |
| Mastodontica | For x 60 kg |
