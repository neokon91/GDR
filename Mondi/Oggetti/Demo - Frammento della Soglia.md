# Demo - Frammento della Soglia

> [!tesoro] Descrizione
> Un frammento di pietra nera attraversato da una vena d'argento. Vibra quando una porta viene chiusa a chiave.

> [!regola] Uso
> Una volta, puo aprire una serratura o sigillare una soglia per una notte.
