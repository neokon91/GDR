# z.bacheche

Bacheche Kanban operative del vault.

| Board | Uso |
| --- | --- |
| `Preparazione Sessioni.md` | Coda di lavoro prima della sessione. |
| `Post Sessione.md` | Consolidamento, canonizzazione e prossima apertura. |
| `Bacheca Creature.md` | Idee e stato delle creature. |

Le bacheche sono Markdown con frontmatter `kanban-plugin: board`. Non usarle come archivio definitivo: le decisioni importanti devono finire in note canoniche dentro `Mondi/`, `Campagne/` o `Risorse/`.
